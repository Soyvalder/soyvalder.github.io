# Soyvalder Portfolio

This project is a personal portfolio website for Valerio Maquera, showcasing skills, projects, and contact information. The portfolio is designed to be responsive and visually appealing, providing an overview of professional experience and capabilities.

## Project Structure

```
soyvalder-portfolio
├── index.html          # Main HTML document for the portfolio
├── script.js           # JavaScript for interactivity
├── package.json        # npm configuration file
├── server.js           # Node.js server setup
├── .gitignore          # Git ignore file
├── README.md           # Project documentation
└── assets
    ├── stylesheet      # CSS files for styling
    │   ├── loader.css  # Styles for loading animation
    │   ├── index.css   # Main styles for the portfolio
    │   ├── tablet.css   # Styles for tablet viewports
    │   └── desktop.css  # Styles for desktop viewports
    ├── images          # Directory for images
    └── pdfs            # Directory for PDF files
```

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Soyvalder/soyvalder-portfolio.git
   cd soyvalder-portfolio
   ```

2. **Install dependencies:**
   Make sure you have Node.js installed. Then run:
   ```bash
   npm install
   ```

3. **Run the server:**
   Start the local server with:
   ```bash
   node server.js
   ```

4. **Access the portfolio:**
   Open your web browser and go to `http://localhost:3000` to view the portfolio.

## Usage

- The portfolio includes sections for Home, About, Skills, Education, Projects, and Contact.
- You can download the CV from the provided link in the Contact section.
